My own processor

This project was all about learning how to use FPGA's and write verilog HDL code. Here you can read the description I copied from Itch.io:
I was interested in how processors and chips are made so I attended a chip design course (tiny tapeout) and then fell in a very deep hole of FPGA's and micro processors. It was only logical to design my own using verilog, making it programmable and testing it on an FPGA board (NandLand GoBoard). After that was done I wanted to add an assembler language and a compiler/simulator for testing my assembler code and compiling it to binary to load it on the FPGA.
