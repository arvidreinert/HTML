My own processor

This project was all about learning how to use FPGA's and write verilog HDL code. Here you can read the description I copied from Itch.io:
I was interested in how processors and chips are made so I attended a chip design course (tiny tapeout) and then fell in a very deep hole of FPGA's and micro processors. It was only logical to design my own using verilog, making it programmable and testing it on an FPGA board (NandLand GoBoard). After that was done I wanted to add an assembler language and a compiler/simulator for testing my assembler code and compiling it to binary to load it on the FPGA. StockDuck

This project was about creating a tool for people to trade stocks with no risks. Here I have copied a description of it, which has been made by ChatGPT, from itch.io. Itch.io:
StockDuck is a lightweight application for creating and managing a virtual stock portfolio with Real Stock prices. It allows you to simulate trading strategies without risking real money, making it ideal for learning, testing ideas, or simply experimenting with the stock market.
Features:
Create and manage a virtual stock portfolio Simulate buying and selling stocks Track portfolio performance over time Useful for strategy testing and learning purposes
Important Note:
This application is still under development, so bugs or unexpected behavior may occur.
